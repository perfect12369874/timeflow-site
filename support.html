入门

我需要每天手动开始 / 停止计时吗？

不需要。开启 HealthKit + Calendar + Motion 权限后，TimeFlow 会被动识别睡眠、会议、移动状态，自动补全你的时间表。手动记录也很快——长按日历空白拖拽即可。

我没有 AI 服务商账号能用吗？

能。AI 洞察是可选功能。即使不连 AI，TimeFlow 也能完整记录、统计、显示趋势。

哪个 AI 服务商最便宜 / 最好？

四家各有特点：Claude 对中文 + 长上下文最稳；Gemini 免费额度大；DeepSeek 价格最低；Kimi 国内访问最友好。在 设置 → AI 提供商 里随时切换。

数据 & 隐私

我的数据存在哪里？

默认全部在你 iPhone 本地的 SwiftData 数据库里。开启 iCloud 同步后，会同步到你私有的 iCloud（Apple 无法读取）。详见 隐私政策。

发给 AI 的数据是什么？

取决于你的隐私级别：L1 只发聚合占比；L2 加匿名时间块；L3 加完整上下文。随时在 设置 → AI 隐私级别 调整。

怎么删掉所有数据？

设置 → 数据与隐私 → 清空所有数据。也可以直接卸载 App —— Keychain 里的 API Key 会一起清除。

能导出数据吗？

能。设置 → 导出 / 备份 选 JSON 或 CSV，通过 Share Sheet 分享或存到 iCloud Drive。

功能

为什么我的睡眠图表是空的？

意味着 HealthKit 没给到睡眠数据。两个原因：
1) 你没有 Apple Watch / iPhone 睡眠追踪：去 健康 app 手动添加，或开启 iPhone 自动睡眠检测
2) 没给 TimeFlow 健康权限：设置 → TimeFlow → 健康 → 允许"睡眠分析"

项目和分类有什么区别？

分类是顶层（睡眠 / 工作 / 学习 / 健身 / 生活 / 反思 / 娱乐），不能改名或删除（但能改 emoji / 颜色）。项目是分类下的子项，由你创建。

能设置目标吗？

每个项目都能。编辑项目 → 打开"设定目标" → 选周期 + 时长。在 洞察 → 目标 看所有项目的进度。
