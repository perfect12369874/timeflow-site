## 常见问题

### 入门

**Q：我需要每天手动开始 / 停止计时吗？**
不需要。开启 HealthKit + Calendar + Motion 权限后，TimeFlow 会被动识别睡眠、会议、移动状态，自动补全你的时间表。手动记录也很快——长按日历空白拖拽即可。

**Q：我没有 AI 服务商账号能用吗？**
能。AI 洞察是可选功能。即使不连 AI，TimeFlow 也能完整记录、统计、显示趋势。

**Q：哪个 AI 服务商最便宜 / 最好？**
四家各有特点，给个粗略参考：
- **Claude**（Anthropic）：对中文 + 长上下文分析最稳，价格中等
- **Gemini**（Google）：免费额度大，速度快
- **DeepSeek**：价格最低，推理能力强
- **Kimi**（Moonshot）：中文 + 国内访问友好

可以在 设置 → AI 提供商 里随时切换。

### 数据 & 隐私

**Q：我的数据存在哪里？**
默认全部在你 iPhone 本地的 SwiftData 数据库里。如果开启 iCloud 同步，数据通过你私有的 iCloud 同步到其他 Apple 设备。详见 [隐私政策](privacy.md)。

**Q：发给 AI 的数据是什么？**
取决于你的 AI 隐私级别：
- **L1**（默认）：只发聚合占比 / 时长统计
- **L2**：+ 匿名化时间块图案（无项目名 / 无位置）
- **L3**：+ 完整上下文（含项目名、会议标题）

随时在 设置 → AI 隐私级别 调整。

**Q：怎么删掉所有数据？**
设置 → 数据与隐私 → 清空所有数据。也可以直接卸载 App —— Keychain 里的 API Key 会一起清除。

**Q：能导出数据吗？**
能。设置 → 导出 / 备份 → JSON 或 CSV，通过 Share Sheet 分享或存到 iCloud Drive。

### 功能

**Q：睡眠债务 (-219.5h) 是什么意思？**
你过去 30 天的睡眠目标 × 天数 = 期望睡眠总时长，减去实际记录的睡眠 = 睡眠债务。如果数字很大，可能是数据不全 —— 检查 设置 → 数据源 → 睡眠与健康 是否已启用。我们会在下一版中改进这个指标的展示方式。

**Q：为什么我的睡眠图表是空的？**
意味着 HealthKit 没给到睡眠数据。两个原因：
1. 你没有 Apple Watch / iPhone 睡眠追踪：去 健康 app 手动添加，或开启 iPhone 自动睡眠检测
2. 没给 TimeFlow 健康权限：设置 → TimeFlow → 健康 → 允许"睡眠分析"

**Q：怎么修改已记录的时间块？**
点一下时间块即可弹出编辑面板，调整时间、分类、项目。

**Q：项目和分类有什么区别？**
分类是顶层（睡眠 / 工作 / 学习 / 健身 / 生活 / 反思 / 娱乐），不能改名或删除（但能改 emoji / 颜色）。项目是分类下的子项，由你创建 —— 比如「工作」分类下可以有「Project Alpha」「SysOps Migration」两个项目。

**Q：能设置目标吗？**
每个项目都能。编辑项目 → 打开"设定目标" → 选周期（每天 / 每周 / 每月 / 每年 / 间隔 N 天）+ 时长。在「洞察 → 目标」可以看到所有项目的进度。
